# SampleDeepLinking
The very basics of deep links in Android apps.

## Instructions
- Install the app
- Open html file from html folder in the browser on mobile device 
- Click on the link
- The app will open 

## Requirements
* Android Studio 3.6 Canary 5
* Kotlin 1.3.41
* Android Gradle Plugin 3.6.0-alpha05
* Gradle wrapper 5.5
* AAPT 2
